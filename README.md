# 🚗 Car Accessory Inventory Manager

Консольное приложение для управления складом автомобильных аксессуаров. 
Позволяет вести учет, изменять цены и формировать отчеты для бизнеса.

## ✨ Основные возможности
* **Полный цикл CRUD:** Добавление, просмотр, обновление и удаление товаров в БД.
* **SQL Integration:** Надежное хранение данных в SQLite с защитой от инъекций.
* **Умный Логгер:** Система декораторов для автоматического аудита всех действий и обработки ошибок.
* **Экспорт в Excel (CSV):** Генерация готовых отчетов с поддержкой кодировки UTF-8 BOM для Windows.
* **Красивый интерфейс:** Табличный вывод данных в консоль с помощью библиотеки `tabulate`.

2. Установите зависимости:
    ```bash
